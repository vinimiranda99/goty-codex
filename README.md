GOTY CODEX é uma enciclopédia interativa de jogos, projetada para quem deseja explorar, descobrir e organizar informações sobre os candidatos ao premio "Game of the Year" (GOTY) 2025. 
A ferramenta funciona como um banco de dados visual, permitindo ao utilizador descobrir jogos de qualidade que talvez ainda não conheça

  - Funcionalidades Chave -
• Visualização de Jogos: A página principal exibe os jogos em formato de "cards", contendo informações como título, desenvolvedora, descrição, plataformas e trailer.
• Os usuários podem navegar pelas categorias nomeadas, que os levam a uma lista filtrada de jogos que se encaixam nessas premiações.
• Filtro por Tags: É possível explorar jogos através de tags que descrevem seus gêneros e características (como "Aventura", "Indie", "Multiplayer"), oferecendo um escopo restrito de busca.
• Busca Direta: Uma barra de pesquisa permite que os usuários procurem por um jogo específico pelo nome.
• Descoberta Aleatória: A função "Descubra um jogo" oferece uma maneira divertida de encontrar um novo título para jogar de forma aleatória. Feita com o objetivo de apresentar jogos novos fora da "bolha" de interesse do usuário
• Sistema de Favoritos: O usuário pode marca seus jogos preferidos, criando uma lista pessoal de fácil acesso.

Em suma, o GOTY CODEX é uma ferramenta para entusiastas de games que querem conhecer ou manter um registro organizado e acessível dos jogos mais aclamados do ano.
